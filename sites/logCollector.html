Log Collector

Technologies used

Backend: Python with Flask
Frontend: HTML with Jinja, CSS, JavaScript
Contenerisation: Docker
Database: MongoDB

Purpose

Application collects and holds log files, and separats found issues, so programmer can close them when they are resolved.
App can be accessed and managed via provided browser GUI or RESTful API.
